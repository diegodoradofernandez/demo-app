Todas las imagenes utilizadas en este sitio son obtenidas de Unsplash API y están publicadas en Unsplash.

Tecnología utilizada

En el desarrollo de este proyecto, se han utilizado las siguientes tecnologías.

  • Angular2
  • Bootstrap
  • jquery

APIS públicas utilizadas

En el desarrollo de este proyecto, se han utilizado las siguientes api publicas.

  • google maps, para la localización de tiendas
  • unsplash para la obtención de imágenes en base a unos parametros de búsqueda
  • qrng de la universidad nacional de australia para la obtención de números aleatorios

    Este servicio web es un poco irregular. A veces tarda en generar los datos númericos, pero acaba hacíendolo.
